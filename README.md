# LNMIIT Library Management System

Welcome to the **Book Management Module** for the LNMIIT Library System. This module is a comprehensive, full-stack application designed to provide a secure, visually humanized, and highly functional interface to systematically manage library literature.

## Features
- **Book Catalog:** Seamlessly view all library assets.
- **RESTful Architecture:** Utilizes robust GET/POST/PUT/DELETE routing algorithms.
- **Advanced Filtering and Search:** Filter books instantly by Category and Author, or use the real-time search utility mapping across Titles, Authors, and ISBN fields natively.
- **Dependency-Free Frontend:** Zero bloated CSS frameworks or external iconography scripts. Native HTML tables and CSS variables create a fast, accessible, enterprise SaaS aesthetic.
- **Non-blocking Backend:** Utilizes a Node.js runtime environment running an asynchronous Express framework connected to a MongoDB document-oriented database.

## Technology Stack
- **Frontend:** Pure HTML5, CSS3, and Vanilla JavaScript (ES8+).
- **Backend:** Node.js, Express.js.
- **Database:** MongoDB internally navigated via Mongoose ODM.
- **Tools:** REST API, async/await native Fetch queries.

## Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) and [MongoDB](https://www.mongodb.com/) vertically installed on your local machine.

### Installation Execution
1. Clone the repository natively.
2. Navigate to the project root internally.
3. Install the raw Node package dependencies:
   ```bash
   npm install
   ```
4. Create a `.env` file mechanically mimicking this setup securely:
   ```env
   PORT=5000
   MONGODB_URI=mongodb://localhost:27017/book_management
   ```
5. Initiate the native Node tracking environment:
   ```bash
   npm start
   ```
6. Open your standard networking browser securely to: `http://localhost:5000`
